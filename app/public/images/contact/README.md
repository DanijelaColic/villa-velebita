# Fotografija za kontakt sekciju

Stavite **jednu** sliku u ovaj folder. U `Contact.tsx` je zadano `OWNER_PHOTO_SRC` — uskladite ime datoteke, npr.:

- **`vlasnici.jpg`**
- **`vlasnici.jpg.jpeg`** (ako sustav doda dupli nastavak)

Ili **`vlasnici.webp`** uz promjenu putanje u komponenti.

Preporuka: širina oko **1000–1600 px**, omjer blizu **4:3** ili portret, format JPG ili WebP.
